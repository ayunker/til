# New Ways to Walk Through Past Commits in git

Today I Learned that `HEAD~` is equivalent to `HEAD~1`, `HEAD~~` is equivalent
to `HEAD~2`, and so on.

This is great - you wouldn't believe the number of times I mistype `git show
HEAD~1`.
